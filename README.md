# Proyecto Red Local - Mercanzapo 

Este proyecto forma parte del módulo de **Redes Locales (0225)**. Se ha diseñado e implementado una infraestructura de red completa para la empresa **Mercanzapo**, centrada en la seguridad y la segmentación de departamentos.

## Características del Diseño
- **Segmentación por VLANs:** Separación lógica entre la red Corporativa (VLAN 10) y la zona de Invitados/Común (VLAN 20).
- **Seguridad (ACL):** Implementación de Listas de Control de Acceso para impedir que usuarios externos accedan a la impresora y servidores privados.
- **Servicios:** Configuración de Servidor Web/DNS interno y salida a Internet (WAN).

## Tecnologías utilizadas
- **Cisco Packet Tracer** (Simulación)
- **Topología:** Estrella extendida.
- **Direccionamiento:** IPv4 Clase C (Subnetting).
